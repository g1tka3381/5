## 1. Алгоритм блок-схема ##
### Алгоритм ###
1. Начало
2. Задать исходные данные:
   * ``` a, b ``` - команды.
3. ЧВыяснить четность/нечетность чисел:
   * ``` (a % 2 == 0 && b % 2 != 0) || (a % 2 != 0 && b % 2 == 0) ```
4. Вывести результат.
5. Конец
